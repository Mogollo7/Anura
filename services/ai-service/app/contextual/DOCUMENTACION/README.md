# Ubicaciones de los Sapitos

En esta carpeta puedes guardar archivos (CSV, TXT, GeoJSON, etc.) que contengan las coordenadas o nombres de los lugares donde se han encontrado las especies de ranas identificadas.

## Sugerencias de organización:
- `coordenadas.csv`: Lista de latitud/longitud por especie.
- `mapas/`: Subcarpeta para archivos de mapas.
- `descripciones/`: Notas sobre el hábitat local.
